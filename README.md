# ppt2video
小米坡PPT转MP4视频工具，自动提取文案配音字幕，支持背景音乐，支持配音文案人工优化。 
# 主要功能 
* 1、PPT文档转MP4视频，兼容office和wps 
* 2、自动从PPT文档提取文案当配音字幕 
* 3、支持人工校验修改配音文案 
* 4、支持添加背景音乐
* 5、支持添加片头片尾、视频背景
* 6、支持批量PPT转MP4视频
